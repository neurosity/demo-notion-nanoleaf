# Neurosity Crown + Nanoleaf

## 1. Install dependencies

Run:

`npm install`

## 2. Get token

Run:

```
npm run token
```

> Make sure to copy the environment variables into a `.env` file.

## 3. Set ENV vars:

```
NANOLEAF_HOST=x.x.x.x
NANOLEAF_ACCESS_TOKEN=?
NEUROSITY_DEVICE_ID=?
NEUROSITY_EMAIL=?
NEUROSITY_PASSWORD=?
```

# 4. Run the sample

Run:

`npm start`

or

`node ./src/calm.js`

or

`node ./src/signalQuality.js`
